## Descrição

Os scripts check-*.sh na pasta /scripts cresceram para 8+ arquivos bash
com duplicação de padrões (cores, resultado, grep). A manutenção está
frágil: heredocs bugam, variáveis não escapadas, e cada script reinventa
o boilerplate. Migrar para um task runner moderno (Taskfile.dev ou mise)
daria configurabilidade, documentação automática, e melhor DX.

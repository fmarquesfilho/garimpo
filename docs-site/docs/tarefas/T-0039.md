## Descrição

Usuários com preferência de dark mode não conseguem usar o app confortavelmente
à noite ou em ambientes escuros. Em telas OLED, dark mode reduz consumo de
energia em até 60%. A paleta dourada/rosa do Garimpei é ideal para dark mode
quando adaptada (não invertida) — tons quentes se destacam em fundos escuros.

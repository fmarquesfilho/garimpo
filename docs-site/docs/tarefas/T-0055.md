## Descrição

O input de busca hoje é texto livre sem indicação de escopo. Com o parser de
tokens, o usuário compõe buscas complexas de forma natural (como CLI/Slack):
`serum @lebotanic #beleza !shopee`. Melhora descobribilidade e reduz cliques.

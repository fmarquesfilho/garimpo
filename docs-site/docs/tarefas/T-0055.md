## Descrição

✅ IMPLEMENTADO e em produção (2026-07-12) como a feature "Omnibox" — spec
`.kiro/specs/omnibox-input/`, merge em main (commit `3a0f938`), deploy verde.

O input de busca era texto livre sem indicação de escopo. Com o parser de
tokens, o usuário compõe buscas complexas de forma natural (como CLI/Slack):
`serum @lebotanic #beleza !shopee`. Melhora descobribilidade e reduz cliques.
O omnibox SUBSTITUIU o sistema de raias (lanes) da página Descobrir.

| Status | `done` |

## Descrição

Garantir que todos os serviços emitem logs estruturados com campos
de correlação (owner_uid, job_id, request_id) e métricas de negócio
(coletas/dia, publicações/dia, alertas/dia) consultáveis no Cloud Logging.

## Critérios de aceite

- ✅ "Todos os endpoints C# logam request_id + owner_uid + duração"
- ✅ "Scheduler loga job_id + tipo + duração + resultado em cada execução"
- ✅ "Collector loga marketplace + keyword + total_found por chamada"
- ✅ "Dashboard no Cloud Logging com filtros por owner_uid e tipo de operação"
- ✅ "Alertas GCP configurados para: erro rate > 5%, latência p95 > 5s"

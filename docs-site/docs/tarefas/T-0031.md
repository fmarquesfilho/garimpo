# T-0031: Compartilhamento de credenciais Shopee entre tenants

| Campo | Valor |
|-------|-------|
| Status | `backlog` |

## Descrição

Permitir que um tenant compartilhe suas credenciais Shopee com outro.
Caso de uso: esposa tem a conta de afiliada, admin usa as credenciais dela.
Implementar campo SharedFromUid e opção no step 2 do onboarding.

## Critérios de aceite

- ✅ "TenantConfig.SharedFromUid: uid do tenant que compartilha credenciais"
- ✅ "Step 2 do onboarding: opção 'Usar credenciais compartilhadas' + input email/uid"
- ✅ "Validação: tenant referenciado deve existir e ter credenciais Shopee configuradas"
- ✅ "Coleta do collector usa credenciais do tenant referenciado (se SharedFromUid preenchido)"
- ✅ "Admin pode ver quem compartilha com ele (lista na tela de configuração)"

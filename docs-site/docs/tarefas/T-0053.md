# T-0053: Avaliar alternativas ao GitHub Actions (CI/CD + hosting de git)

| Campo | Valor |

## Descrição

GitHub Actions bloqueou workflows por problema de billing. O projeto
depende 100% do GitHub para git + CI/CD + deploy. Avaliar alternativas
para reduzir risco de vendor lock-in e custo.

## Critérios de aceite

- ✅ "Pesquisar: GitLab CI, Bitbucket Pipelines, Codeberg + Woodpecker, Gitea + Drone"
- ✅ "Comparar: minutos grátis, preço excedente, suporte a Docker, GCP deploy, Cloudflare"
- ✅ "Avaliar GitHub Education (professor = Pro grátis + Actions ilimitados)"
- ✅ "Decisão documentada como ADR com trade-offs"
- ✅ "Se migrar: manter GitHub como mirror (não perder histórico/PRs)"

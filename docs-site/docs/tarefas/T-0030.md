## Descrição

O onboarding (step 3) só oferece Telegram. Para multi-tenant completo,
cada tenant precisa poder configurar WhatsApp (Meta Cloud API) pela interface.
Também precisa fechar o gap: tokens salvos no onboarding devem ser usados
na publicação (depende de T-0027 no backend, mas o frontend pode já estar pronto).

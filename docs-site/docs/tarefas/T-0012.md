## Descrição

Extrair o scheduler para um microserviço Go com cron nativo e
goroutines. Orquestra jobs periódicos: coleta Shopee, export BigQuery,
verificação de alertas. Substitui Cloud Scheduler e Hangfire.

## Critérios de aceite

- ✅ scheduler.proto com TriggerJob, ListJobs, SetSchedule RPCs
- ✅ Servidor Go com cron library (robfig/cron ou similar)
- ✅ Chama collector, publisher, alerter via gRPC interno
- ✅ Configuração de schedules via proto/config
- ✅ Dockerfile multi-stage
- ✅ Health check + métricas de jobs executados

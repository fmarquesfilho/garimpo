## Descrição

Configurar deploy no Cloud Run com multi-container: Web App C# como
container principal (ingress), microserviços Go como sidecars gRPC.
Comunicação interna via localhost.

## Critérios de aceite

- ✅ cloud-run service YAML com multi-container config
- ✅ Container principal (C#) recebe tráfego HTTP externo
- ✅ Sidecars (collector, publisher, alerter, scheduler) expõem gRPC em localhost
- ✅ CI/CD deploya todos os containers atomicamente
- ✅ Health checks para cada container
- ✅ Rollback funcional (revisão anterior)

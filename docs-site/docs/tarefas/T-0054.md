## Descrição

⚠️ SUPERSEDED (2026-07-12): a página Descobrir NÃO usa mais raias — o Omnibox
(T-0055) substituiu as lanes. Os specs locais reescritos aqui miram seletores/
fluxos de raia (abrirRaiaFiltros, abrirRaiaLojas, Combobox de loja/categoria,
Lane.svelte) que não existem mais. Portanto voltaram a ficar obsoletos e
precisam ser reescritos de novo para o Omnibox:
  - digitar keyword → resultados
  - digitar `@`/`#`/`!` + 2 chars → dropdown agrupado; selecionar → card/escopo
  - Enter resolve tokens digitados (keyword + @loja + #cat + !mkt)
  - navegação por teclado (↑/↓/Enter/Esc), degradação sem lojas/categorias
Arquivos: `web/tests/local/*.spec.js` e `web/tests/prod/descobrir.spec.js`.
Fora dos gates exigidos hoje (test:e2e-novos é backend). Ver handoff omnibox.

--- valor original (raias) ---

O redesign da página Descobrir em raias (PR #42, ADR-0004) mudou a UI e os
fluxos. Os E2E locais existentes miram a UI antiga e falham. Como não rodam na
CI, o merge não é bloqueado — mas a rede de segurança da página core fica
descoberta até serem reescritos.

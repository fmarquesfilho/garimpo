## Descrição

Astro Starlight é pesado e desconectado da stack (Svelte/Vite). Migrar para
VitePress que usa Vite nativo, é mais leve, tem search integrado, e deploy
trivial no Cloudflare Pages. Alternativa: KitDocs (SvelteKit-native) se quiser
manter 100% da stack em Svelte.

## Critérios de aceite

- ✅ "docs-site funciona com VitePress (ou KitDocs)"

- ✅ "Build <10s (VitePress é ~2-3s para sites pequenos)"
- ✅ "Deploy no Cloudflare Pages via workflow docs.yml"
- ✅ "OpenAPI spec renderizada (swagger-ui ou plugin VitePress)"
- ✅ "Search integrado funcionando"

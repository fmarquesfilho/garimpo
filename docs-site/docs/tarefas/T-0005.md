## Descrição

Hoje alertas estão desabilitados por falta de configuração por tenant.
Cada usuário deve poder definir threshold, canais e categorias. Após ADR-0012,
config salva em PostgreSQL (C#), disparo via alerter Go (gRPC).

## Critérios de aceite

- ✅ Tela de configuração de alertas no frontend
- ✅ Config salva por tenant no PostgreSQL (via Web App C#)
- ✅ Alertas disparam automaticamente nas coletas (alerter Go via gRPC)
- ✅ Scheduler Go orquestra a verificação periódica

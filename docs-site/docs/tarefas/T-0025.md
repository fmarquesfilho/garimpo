## Descrição

Serviço Python para queries analíticas no BigQuery (novidades, quedas,
evolução, estatísticas). Substitui a lógica analítica do monólito Go.
Prepara ecossistema para IA/ML (T-0007). ADR-0014.

## Critérios de aceite

- ✅ "FastAPI com endpoints: /novidades, /quedas, /evolucao, /estatisticas, /health"
- ✅ Queries BigQuery portadas do Go (snapshots, variações)
- ✅ C# chama analyzer via HTTP (HttpClient)
- ✅ Dockerfile pronto para deploy como sidecar
- ✅ Docker Compose inclui analyzer

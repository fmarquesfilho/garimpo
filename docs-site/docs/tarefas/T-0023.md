# T-0023: Migrar WhatsApp sender de Maytapi para Meta Cloud API

| Campo | Valor |
|-------|-------|
| Status | `done` |
| Prioridade | alta |
| Estimativa | M |
| Sprint | 2026-S27 |
| Tags | [backend, whatsapp, meta, publish, go-microservice] |

## Descrição

Substituir intermediário Maytapi pela API oficial da Meta. Reduz custo,
remove dependência de terceiro, e habilita features oficiais (templates,
botões interativos). ADR-0013.

## Critérios de aceite

- ✅ WhatsAppSender usa Graph API da Meta (v25.0)
- ✅ "Variáveis de ambiente: WHATSAPP_ACCESS_TOKEN, WHATSAPP_PHONE_NUMBER_ID"
- ✅ Envio de texto e imagem funcionando
- ✅ Multi-grupo mantido (enviar para N destinatários)
- ✅ Factory atualizada (publish/novo.go e services/publisher)
- ✅ Variáveis Maytapi removidas do código
- ✅ Testes com mock do Graph API

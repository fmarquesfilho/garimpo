## Descrição

Completar a separação de responsabilidades: PostgreSQL é a fonte
primária para dados transacionais, BigQuery recebe apenas export
periódico para analytics e métricas históricas.

## Critérios de aceite

- ✅ Todas as rotas leem/escrevem em PostgreSQL
- ✅ Export periódico PG → BQ para analytics (via scheduler Go)
- ✅ Conversões permanecem direto no BQ (write pelo scheduler)
- ✅ Dashboard/analytics consultam BQ
- ✅ Dual-write removido (fase final)

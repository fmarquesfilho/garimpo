## Descrição

A spec OpenAPI em api/openapi.yaml está desatualizada e não inclui os
endpoints gRPC (Collector, Publisher, Scheduler). Deve ser regenerada
automaticamente a cada build (se houver mudança) e servir um Swagger UI
acessível em /docs/api ou via Scalar.

## Critérios de aceite

- ✅ "OpenAPI spec gerada automaticamente a partir do código C# (Minimal API metadata)"
- ✅ "Endpoints gRPC documentados via proto → OpenAPI (buf ou grpc-gateway annotations)"
- ✅ "Swagger UI (Scalar) acessível em https://garimpei.app.br/docs/api"
- ✅ "Regeneração condicional no CI: só se src/ ou protos/ mudaram"
- ✅ "Spec inclui: endpoints HTTP (/api/*), gRPC services (Collector, Publisher, Scheduler)"
- ✅ "Versionamento: spec commitada em api/openapi.yaml (fonte de verdade)"
- ✅ "check:api-contract valida contra a spec gerada"

## Descrição

Isolar dados por tenant (owner_uid). Pré-requisito para onboarding de novos
afiliados. Após ADR-0012, será implementado em C# com EF Core + PostgreSQL
(global query filters por tenant). Candidata natural para primeira feature no
Web App C#.

## Critérios de aceite

- ✅ Queries filtram por owner_uid automaticamente (EF Core global filter)
- ✅ Testes verificam isolamento entre tenants
- ✅ Buscas e eventos vinculados ao owner
- ✅ Implementado no Web App C# (PostgreSQL)

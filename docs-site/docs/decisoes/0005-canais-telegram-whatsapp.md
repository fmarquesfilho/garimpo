# ADR 0005 — Canais de publicação: Telegram + WhatsApp

## Contexto

Docs mencionavam Instagram como canal ativo. Na prática, apenas Telegram e
WhatsApp (via Maytapi) estão implementados.

## Decisão

Canais ativos: **Telegram** e **WhatsApp**.
Instagram é **visão futura** — toda menção como canal ativo é aspiracional (🔮).

## Consequências

- Código de publicação só suporta Telegram e WhatsApp.
- Qualquer doc que mencione Instagram como canal funcional deve ser corrigido.
